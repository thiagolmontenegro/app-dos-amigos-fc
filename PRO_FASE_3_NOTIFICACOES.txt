Fase PRO 3 — Notificações inteligentes

Recursos:
- Central de notificações dentro do app
- Histórico salvo em app_notifications
- Notificação ao abrir, bloquear, desbloquear ou remover lista
- Notificação quando atleta altera status
- Alerta quando a lista está a menos de 30 minutos de fechar
- Alerta quando o prazo encerra
- Notificação de movimento de caixa
- Usa também Notification API do navegador se o usuário ativar Push

Rode o SQL em supabase/migrations/pro_fase_3_notificacoes.sql antes de testar.
