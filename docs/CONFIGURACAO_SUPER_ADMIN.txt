CONFIGURAÇÃO DO SUPER ADMIN

1) Na Vercel:
Project > Settings > Environment Variables

Adicione:
VITE_SUPER_ADMIN_EMAIL = seuemail@dominio.com

Depois faça Redeploy.

2) No Supabase Edge Functions:
Edge Functions > Secrets

Adicione:
SUPER_ADMIN_EMAIL = seuemail@dominio.com

A secret SERVICE_ROLE_KEY deve continuar existindo.

3) No banco Supabase:
Garanta que o usuário Super Admin também esteja na tabela players com:
role = admin
approved = true
email = o mesmo e-mail definido acima
user_id = UID do Authentication

Observação:
As policies SQL usam a tabela players/role admin como base principal de permissão.
