# App dos Amigos FC

Projeto React + Vite + Supabase + PWA.

## Rodar local
```bash
npm install
npm run dev
```

## Variáveis de ambiente
Crie um arquivo `.env` na raiz:

```env
VITE_SUPABASE_URL=https://tvouiuulgqoutyvievui.supabase.co
VITE_SUPABASE_ANON_KEY=SUA_ANON_KEY
```

## Deploy na Vercel
- Suba este projeto na Vercel.
- Em Environment Variables, cadastre:
  - VITE_SUPABASE_URL
  - VITE_SUPABASE_ANON_KEY
- Deploy.

## Instalar no celular
Depois do deploy, abra no Chrome Android e toque em "Adicionar à tela inicial".
